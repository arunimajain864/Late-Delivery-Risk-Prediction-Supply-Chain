# Machine Learning–Based Late Delivery Risk Prediction in Global Supply Chain Operations

## Project Overview

This project develops a machine learning framework to predict late delivery risks in global supply chain operations. The objective is to identify high-risk shipments before delays occur, enabling proactive intervention and better logistics planning.

## Problem Statement

Shipment delays result in customer dissatisfaction, SLA violations, increased operational costs, and reduced profitability. Traditional logistics systems identify delays only after they occur. This project focuses on predicting late delivery risks before shipment execution.
